# Taller1-Victor-Sostenibilidad

## Introducción
Este proyecto analiza la **sostenibilidad** aplicada a una consultora de desarrollo web, identificando qué aspectos son críticos tanto para la empresa como para la sociedad.

## Investigación Rigurosa
Para este análisis se han consultado fuentes en **Dialnet** y **arXiv**, enfocando la búsqueda en los términos *Green IT* y *Materiality in ICT*.

### Grupos de Interés (Stakeholders)
1. **Usuarios Finales**: Afectados directamente por la gestión de la *privacidad*.
2. **Desarrolladores**: Clave en la implementación de un código eficiente.
3. **Proveedores Cloud**: Responsables de la *huella de carbono* del servidor.

### Temas Materiales (ASG)
**Ambiental**: Eficiencia energética del código para reducir el impacto ambiental.
**Social**: Accesibilidad universal para garantizar la inclusión digital.
**Gobernanza**: Transparencia algorítmica y protección contra la corrupción.
